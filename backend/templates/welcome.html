Open CryptoStego

CryptoStego is a method aimed at amalgamating both the cryptography and steganography techniques for a better information security. The primary purpose of this tool is to build up a new method of hiding secret text messages inside an image, by combining both cryptography and steganography. So a new algorithm is proposed and implemented to achieve this.
